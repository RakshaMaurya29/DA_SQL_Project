# Music Store SQL Analysis

This project was completed as part of a SQL learning exercise inspired by a YouTube tutorial.
I independently wrote and executed the SQL queries, analyzed the results, and documented the insights.

## Project Overview

This project analyzes a music store database using SQL to uncover customer purchasing behavior, sales trends, artist performance, and genre popularity.

## Project Questions

### Easy
1. Senior most employee
2. Countries with highest invoices
3. Top invoice values
4. Best customer city
5. Best customer

### Moderate
1. Rock music listeners
2. Top Rock artists
3. Songs longer than average

### Advanced
1. Customer spending by artist
2. Most popular genre by country
3. Top customer by country

# Business Insights

## Customer Analysis

- USA generated the highest number of invoices.
- Customers from major cities contribute most revenue.

## Revenue Analysis

- Certain customers account for a significant portion of sales.
- Promotional campaigns can target high-value customers.

## Genre Analysis

- Rock is one of the most purchased genres.
- Rock listeners represent a valuable customer segment.

## Artist Analysis

- Top artists contribute heavily to overall music purchases.
- Popular artists should be prioritized for marketing campaigns.
